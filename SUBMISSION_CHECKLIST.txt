VIVA PREPARATION TIPS:
======================

Be ready to explain:

1. CAESAR CIPHER
   - Formula: E(x) = (x + k) mod 26
   - Weakness: Only 26 possible keys (brute force attack)

2. VIGENÈRE CIPHER
   - Uses repeating keyword for different shifts
   - Vulnerable to Kasiski examination

3. PLAYFAIR CIPHER
   - 5×5 matrix, I/J combined
   - Three rules: same row, same column, rectangle

4. HILL CIPHER
   - Matrix multiplication mod 26
   - Key must be invertible (gcd(det, 26) = 1)

5. AES-128
   - Block size: 128 bits, Key: 128 bits, Rounds: 10
   - SubBytes, ShiftRows, MixColumns, AddRoundKey

6. DES
   - Block: 64 bits, Key: 56 bits, Rounds: 16
   - Feistel structure, S-boxes, Permutations
   - WHY INSECURE: 56-bit key can be brute-forced

7. DIFFIE-HELLMAN
   - A = g^a mod p, B = g^b mod p
   - Shared secret K = g^(ab) mod p
   - Security: Discrete Logarithm Problem

8. EXTRA FEATURES (Be ready to demonstrate!)
   - Load from File: Supports multiple encodings (UTF-8, UTF-16, etc.)
   - Save to File: Exports encrypted/decrypted results
   - OCR: Uses Tesseract to extract text from images
